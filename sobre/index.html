Vetor de Acessibilidade para Neurodiversidade e Experiência

O VANE é um guia de recomendações voltado ao desenvolvimento de softwares e interfaces digitais mais inclusivas, com o objetivo de apoiar profissionais na criação de experiências acessíveis para pessoas com Transtorno do Espectro Autista (TEA), especialmente no contexto de uso de dispositivos multitoque. O nome do projeto é uma sigla para Vetor de Acessibilidade para Neurodiversidade e Experiência, refletindo seu propósito de orientar práticas de design e desenvolvimento centradas no usuário. Além disso, o nome também carrega um significado pessoal, sendo uma homenagem à minha avó Ivanildes, carinhosamente chamada de Vane. As recomendações apresentadas no guia combinam orientações técnicas com exemplos visuais e práticos. Embora algumas abordagens sejam mais detalhadas do ponto de vista técnico, a proposta do VANE é ser acessível também a pessoas que não possuem formação em tecnologia, mas que participam do desenvolvimento de projetos digitais voltados a indivíduos no espectro autista.

GAIA

O GAIA é um guia de acessibilidade de interfaces web com foco em aspectos do autismo, que ajudou como base e referência para o guia.

Sobre o Projeto

Este projeto visa disseminar o conhecimento sobre acessibilidade digital, com foco em inclusão para todas as pessoas, incluindo aquelas com Transtorno do Espectro Autista (TEA).
