VANE -Vetor de Acessibilidade para Neurodiversidade e Experiência

Guia para um Design Inclusivo com Foco em TEA, acesse boas práticas para criar experiências digitais inclusivas, para pessoas no Transtorno do Espectro Autista (TEA) e para todos.

Sobre o Projeto

Este projeto visa disseminar o conhecimento sobre acessibilidade digital, com foco em inclusão para todas as pessoas, incluindo aquelas com Transtorno do Espectro Autista (TEA).
